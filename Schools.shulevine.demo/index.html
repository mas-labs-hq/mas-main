Load Data

Bulk import student data from a spreadsheet. No need to type students one by one.

How it works:

  1. Click Download Template to get a blank CSV with the right column headers
  2. Open it in Excel or Google Sheets, paste your student data into the columns
  3. Save the file as CSV (Comma Separated Values)
  4. Click Import from CSV and select your filled template

If your data is in a Word document or PDF, copy the table into the template in Excel or Google Sheets, then save as CSV and import.

Stream names: You can type any stream name in the Stream column (e.g. North, South, Alpha, East, West). It does not have to be A, B, C.
